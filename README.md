## 📌 [Snippetbox](https://github.com/Nikita-Solokha/snippetbox)

Веб-приложение для хранения и управления текстовыми сниппетами (заметками, фрагментами кода).

### 🔧 Стек
Go, PostgreSQL, HTML/CSS, Bootstrap, Git, Linux, pgAdmin

### ✅ Функционал
- CRUD-операции: создание, просмотр, удаление сниппетов  
- Хранение данных в PostgreSQL через `database/sql`  
- Шаблонизация с `html/template` + кэширование шаблонов  
- Чистая архитектура: разделение логики, шаблонов и работы с БД  
- Логирование (info, error)  
- Минималистичный интерфейс на Bootstrap  
- Базовая защита от уязвимостей (XSS, SQL-инъекции)

## 🔐 [JWT Auth API](https://github.com/Nikita-Solokha/jwt-go)

Простой REST API с аутентификацией на основе JWT (JSON Web Tokens).

### 🔧 Стек
Go, JWT, REST API, JSON, Gorilla Mux (или другой роутер, если использовал)

### ✅ Функционал
- Регистрация и вход пользователя  
- Генерация и проверка JWT-токенов  
- Защита приватных эндпоинтов с помощью middleware  
- Валидация входных данных  
- Ответы в формате JSON

> 💡 Проект помог мне глубже понять механизмы аутентификации, работу с токенами и безопасность API.

👉 [Открыть репозиторий](https://github.com/Nikita-Solokha/jwt-go)
